OpenComments Example

Basic Usage

OpenComments will automatically initialize when the script loads.

Look for the widget in the top-left corner of the page.

Custom Configuration

You can configure the API URL and initialization:


<script>
  // Configure before script loads
  window.OpenComments = {
    config: {
      apiUrl: 'https://api.example.com',
      autoInit: true
    }
  };
</script>
<script src="dist/opencomments.js"></script>

Manual Initialization

Or initialize manually:


<script src="dist/opencomments.js"></script>
<script>
  OpenComments.init({
    apiUrl: 'https://api.example.com'
  });
</script>
